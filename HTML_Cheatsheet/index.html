HTML/CSS Cheatsheet

Base HTML Template Tags

All HTML pages require these basic tags to form the traditional 'web page':

Tag Name Description
<html> HTML This tag begins and ends the file containing all of the page content within it.
<head> HEAD Denotes the Head section of the page often contains CSS links and other references.
<body> BODY Contains the main displayed content of the web page. This is the "meat" of your web page.

Closing HTML Tags

All tags used within an HTML document must have a corresponding closing tag, allowing the browser to accurately interpret the proper layout of the page content.

Tag Name Description
</html> Closing HTML Tag This tag closes the HTML content and is typically the last element written on the file.
</head> Closing HEAD Tag Closes the Head section of the page.
</body> Closing BODY Tag Closes the main section "Body" of the web page.

NOTE: We have not included the <!DOCTYPE> in these tables as it is not a tag/element. It is only an instruction to the browser about what the document type is. It is considered a Null Element and has no closing tag as it cannot contain any content.
